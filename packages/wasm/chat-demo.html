🚀 Chat App Demo

This is a demonstration of the BuzzU Chat App UI. In a real implementation:

  • ✅ The WASM module provides secure crypto operations
  • ✅ WebSocket signaling runs through the Rust WSS worker
  • ✅ WebRTC peer-to-peer connections handle real-time messaging
  • ✅ Connection quality monitoring optimizes performance
  • ✅ All heavy operations run in Web Workers to prevent UI blocking

To implement this for real:

  1. Build the WASM: wasm-pack build --target web
  2. Import the ChatApp component: import { ChatApp } from './js-bridge/chat-app'
  3. Set up a WebSocket signaling server (see workers/signaling/)
  4. Deploy and enjoy your Rust-powered chat app!
